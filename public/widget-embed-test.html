Prueba del widget

Obligatorio: la clave son unos 64 caracteres (hex). Cópiala del panel Configuración → Widget en tu web → Copiar clave. Si dejas TU_CLAVE_PUBLICA, el chat dará error 401.

Opción A: edita esta página y pega tu clave en data-site-key="…". Opción B: abre ?key=PEGA_AQUI_TU_CLAVE_COMPLETA en la URL (sin espacios).

Si solo pegas la URL https://wazapp.ai/widget.js en el editor, el navegador mostrará código fuente: hace falta la etiqueta <script src="..."> como abajo.
